Category | Methods About Category

/category/ Retorna todas as categorias habilitadas
/category/disabled Retorna todas as categorias desabilitadas
/category/id/{id} Retorna a categoria de acordo com o { id }
/category/title/{title} Retorna uma ou várias categorias, de acordo com o { title }
/category/{id}/subcategory Retorna as subcategorias que pertencem a categoria, de acordo com o { id }
/category/register Cadastra uma categoria
/category/{id} Altera os dados da categoria de acordo com o { id }
/category/{id} Deleta a categoria de acordo com o { id }
